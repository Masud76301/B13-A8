# QurbaniHat

## Purpose 

A modern livestock marketplace where users can explore animals for Qurbani such as cows and goats. Users can  also book livestock for qurbani and get ideas on a variety  of certified animals from the best local  farms. This platform aims to provide high-quality  Qurbani animals at reasonable prices , along with reliable doorstep delivery  for a seamless and convenient experience.

## Live URL 
https://b13-a8-beryl.vercel.app


## Key Features 
- Filter & Sort: Full animal list with filter by type (Cow, Goat, Sheep) and sort by price
- Profile: Complete animal profile including weight, age, farm info, certifications, and a booking form 
- Booking Form :  Name, email, phone, date, and delivery address fields with a success toast confirmation 
- Authentication:  Login and Register pages; booking is restricted to logged-in users only 
- Responsive: the website is fully responsive on mobile, tablet, and desktop
